## ğŸ” Quick Reference

### Common Tasks

- **Open console**: Press `~`
- **Teleport player**: `player.teleport X Y`
- **Load map**: `map.load mapname`
- **Open profiler**: Press F2 or `tab profiler`
- **Open event inspector**: Press F4 or `tab eventinspector`
- **Reload script**: Edit and save - auto-reloads!

### Key Concepts

- **ECS** - Entity Component System for game objects
- **Event Bus** - Decoupled communication between systems
- **Hot Reload** - Edit code without restarting
- **Component Pooling** - Reuse objects for performance
- **Spatial Hashing** - Fast collision detection
